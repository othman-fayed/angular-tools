Welcome to Angular Tools

This application demonstrates the Dynamic Side Menu components built with Angular {{ getAngularVersion() }} and Bootstrap 5.

Features:
  • Dynamic side menu with configurable options
  • Support for nested menu items
  • Bootstrap 5 styling and icons
  • Collapsible menu functionality
  • Badge support for notifications
  • Light and dark theme support
  • Responsive design
Click on menu items in the sidebar to see the dynamic behavior. Items with children will expand/collapse.
